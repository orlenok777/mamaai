# MamaAI Health Vision 2.0

Полностью веб-приложение: ничего не нужно устанавливать на компьютер.

## Возможности

- камера работает прямо в браузере;
- кадры анализируются Google Gemini;
- AI определяет предметы в руках и действия возле рта;
- при воде запускается голосовой диалог и записывается количество миллилитров;
- при лекарстве AI пытается прочитать название упаковки;
- если AI ошибся, пользователь диктует правильное название;
- журнал воды и лекарств хранится в браузере и экспортируется в CSV;
- приложение публикуется через GitHub Pages.

## Запуск

Откройте сайт GitHub Pages в Google Chrome. Вставьте Gemini API key, нажмите «Подключить Google AI», затем «Запустить помощника» и разрешите камеру и микрофон.

Ключ можно хранить только до закрытия вкладки или запомнить в конкретном браузере. Он не записывается в репозиторий.

## Безопасность

Название и факт приёма лекарства всегда подтверждаются голосом. Приложение не назначает лечение и не заменяет врача.
